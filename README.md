# Template App

**Phonegap** template for create projects with npm libraries that uses *ES6* Features (**JavaScript**)

+ `git clone git@10.25.253.3:DPD/phonegap-template.git`
+ Edit **package.json** => 
   + **name**
   + **description**
   + **author**
+ Edit **config.xml** =>
   + `<name>`
   + `<description>`
   + `<author>`
